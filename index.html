About the Project

This project was inspired by a personal need to manage and track my finances more effectively. The idea came during my time at Holberton School, where we were encouraged to build practical solutions to real-world problems. This is a Portfolio Project for Holberton School.

Check out the [Holberton School](https://www.holbertonschool.com/) for more information.
